ideapad15arr-opencore

An effort to boot macOS on my personal laptop [repo]. The EFI folder is confirmed to partially work on Lenovo Ideapad 330-15ARR for macOS Ventura 13.7.6. Feature support is documented in the README.

Installation

Format a drive as FAT32 and transfer the following folders onto it:

USB
├─ com.apple.recovery.boot/
└─ EFI/

Boot into the Recovery and install macOS as normal.

macOS Recovery images can be retrieved using macrecovery.
