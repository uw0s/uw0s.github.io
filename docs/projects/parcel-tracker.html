parcel-tracker

parcel-tracker [code] is a CLI tool for tracking packages.

Installation

pip install https://api.github.com/repos/uw0s/parcel-tracker/tarball/main#subdirectory=python

Usage

parcel-tracker 0123456789ABCDEF carrier_name
